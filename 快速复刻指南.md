```tsx

  <Button onClick={() => runSpeedtest()} size="xl" fullWidth leftIcon={<Play size={19} />}>

    variant="secondary" 
    size="xl"
```


## 任务2：实现 Reduce Motion + Larger Text（核心，20-30分钟）

### 整体思路
需要改 4 个文件：
- `src/store/vpnStore.ts`（状态 + 持久化）
- `src/App.tsx`（全局属性 + MotionConfig）
- `src/index.css`（视觉效果）
- `src/pages/Appearance.tsx`（UI + 实时预览）

---

### 步骤1：在 Store 添加状态（5分钟）

打开 `src/store/vpnStore.ts`，进行以下修改：

**1.1 在 interface 里添加字段**（Theme 附近）：
```ts
theme: 'dark' | 'light'
reduceMotion: boolean
largerText: boolean
```

**1.2 添加 Action**（setTheme 附近）：
```ts
setReduceMotion: (enabled: boolean) => void
setLargerText: (enabled: boolean) => void
```

**1.3 设置默认值**（theme: 'dark' 下面）：
```ts
theme: 'dark',
reduceMotion: false,
largerText: false,
```

**1.4 实现 Action**（toggleTheme 下面）：
```ts
setReduceMotion: (enabled) => set({ reduceMotion: enabled }),
setLargerText: (enabled) => set({ largerText: enabled }),
```

**1.5 在 partialize 里加上持久化**（theme 下面）：
```ts
theme: state.theme,
reduceMotion: state.reduceMotion,
largerText: state.largerText,
```

---

### 步骤2：在 App.tsx 全局应用（5分钟）

打开 `src/App.tsx`：

**2.1 引入 MotionConfig**：
```tsx
import { MotionConfig } from 'framer-motion';
```

**2.2 修改 useVPNStore 读取**：
```tsx
const { theme, reduceMotion = false, largerText = false } = useVPNStore();
```

**2.3 扩展 useEffect 设置 data 属性**：
```tsx
useEffect(() => {
  const root = document.documentElement;
  root.setAttribute('data-theme', theme);
  root.setAttribute('data-reduce-motion', reduceMotion ? 'true' : 'false');
  root.setAttribute('data-larger-text', largerText ? 'true' : 'false');
  // ... 原来的 theme-light/dark class 逻辑保持不变
}, [theme, reduceMotion, largerText]);
```

**2.4 用 MotionConfig 包裹内容**：
```tsx
<MotionConfig reducedMotion={reduceMotion ? "always" : "user"}>
  <div className="min-h-screen ...">
    <Routes>...</Routes>
  </div>
</MotionConfig>
```

### 步骤3：编写 CSS 规则（8分钟）

在 `src/index.css` 文件**最底部**添加以下内容：

```css
/* ========== Reduce Motion + Larger Text ========== */

@media (prefers-reduced-motion: reduce),
[data-reduce-motion="true"] {
  *, *::before, *::after {
    transition-duration: 0.01ms !important;
    animation-duration: 0.01ms !important;
  }
  .btn, .glass, .glass-strong, .orb, [class*="transition"] {
    transition: none !important;
  }
}

[data-larger-text="true"] {
  --text-scale: 1.2;
  font-size: calc(16px * var(--text-scale));
}

/* 覆盖项目中实际使用的所有字号 */
[data-larger-text="true"] .text-xs,
[data-larger-text="true"] .text-\[10px\] { font-size: calc(0.625rem * var(--text-scale)) !important; }
[data-larger-text="true"] .text-sm,
[data-larger-text="true"] .text-\[13px\] { font-size: calc(0.75rem * var(--text-scale)) !important; }
[data-larger-text="true"] .text-\[15px\] { font-size: calc(1rem * var(--text-scale)) !important; }
[data-larger-text="true"] .text-\[28px\] { font-size: calc(1.75rem * var(--text-scale)) !important; }
[data-larger-text="true"] .font-mono.text-5xl,
[data-larger-text="true"] .text-5xl { font-size: calc(3rem * var(--text-scale)) !important; }
/* 根据自己项目实际用到的字号继续补充 */
```

## 常见问题速查

| 现象               | 解决方案                                      |
|--------------------|-----------------------------------------------|
| 开关点了没反应     | 检查是否加了 `= false` 默认值                 |
| 文字没变大         | CSS 里没覆盖项目实际用的字号，需补充选择器    |
| 动画没减少         | 重启 dev server + 清 localStorage             |
| 刷新后状态丢失     | 确认已在 partialize 里写入新字段              |
| 预览区域没变化     | 检查 Toggle 的 onCheckedChange 是否正确调用 setter |
